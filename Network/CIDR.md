### CIDR
```
클래스가 없는 도메인 간 라우팅기법
```

### 기존 IP ADDRESS
```
network address : 네트워크 식별
host address : 네트워크 내 호스트 컴퓨터 식별
클래스 기반 아이피 주소 할당
```

### cidr의 장점
```
기존 방식보다 유연하게 비트단위로 network address와 host address를 나눌 수 있다.
이를 통해 증가한 ip address요구를 충족시킬 수 있다.
```
