### UDP란?
```
- user datagram protocol의 약자로 데이터그램 단위로 처리하는 프로토콜이다.
- ★비연결형, 신뢰성 없는 전송 프로토콜
- transport layer
```
### UDP 사용이유
```
- udp의 가장 큰 장점은 데이터의 신속성
- 영상을 볼 때 부분부분 조금 깨지는것이 완벽한 영상을 받기위해 버퍼링이 걸리는것보다 낫다.
```
