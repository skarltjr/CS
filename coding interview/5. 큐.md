
### ps 1
```
큐가 제공하는 오퍼레이션만 사용해서 큐를 뒤집는 코드를 작성하라.
제공하는 오퍼레이션은 다음과 같다.
offer(E e): 큐의 끝에 데이터를 추가한다.
E poll(): 큐 헤드를 꺼낸다.
isEmpty(): 큐가 비어있는가.

예) Q {1, 2, 3}  =>  Q {3, 2, 1}

```
```
private static void reverse(Queue<Integer> q) {
    if (q.isEmpty()) {
        return;
    }

    Integer temp = q.poll();
    reverse(q);
    q.add(temp);
}
```
