```
제공하는 오퍼레이션은 다음과 같다.
offer(E e): 큐의 끝에 데이터를 추가한다.
E poll(): 큐 헤드를 꺼낸다.
isEmpty(): 큐가 비어있는가.

예) Q {1, 2, 3}  =>  Q {3, 2, 1}

```
```
private static void reverse(Queue<Integer> q) {
    if (q.isEmpty()) {
        return;
    }

    Integer temp = q.poll();
    reverse(q);
    q.add(temp);
}
```

- lru cache
```
class MyLRUCache{
    int size;
    Queue<Integer> q;

    public MyLRUCache() {
        this.size = 10;
        this.q = new LinkedList<>();
    }

    public void add(Integer target) {
        if (q.contains(target)) {
            q.remove(target);
            q.add(target);
            return;
        }


        if (q.size() >= size) {
            q.poll();
        }
        q.add(target);
    }

    public Integer poll() {
        if (q.isEmpty()) {
            throw new IllegalArgumentException("q is empty");
        }
        return q.poll();
    }

    public Integer peek() {
        if (q.isEmpty()) {
            throw new IllegalArgumentException("q is empty");
        }

        Integer poll = q.poll();
        q.add(poll);
        return poll;
    }

    public void see() {
        for (Integer integer : q) {
            System.out.print(integer + " ");
        }
    }
}
```
