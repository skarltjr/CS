## stack - array로 구현
```
class MyStack {
    int size;
    int[] arr;
    int topIndex;

    public MyStack(int size) {
        this.size = size;
        this.arr = new int[size];
        this.topIndex = -1;
    }

    public void push(int val) {
        if (topIndex == size - 1) {
            return;
        }

        arr[++topIndex] = val;
    }

    public int pop() {
        if (topIndex == -1) {
            return 'x';
        }

        return arr[topIndex--];
    }

    public int peek() {
        if (topIndex == -1) {
            return 'x';
        }

        return arr[topIndex];
    }
}
```
## stack - list로 구현
```
class Node {
    int val;
    Node prev;
    Node next;

    public Node(int val) {
        this.val = val;
        this.prev = null;
        this.next = null;
    }
}
class MyStack {
    Node head;

    public MyStack() {
        this.head = null;
    }

    public void push(Node node) {
        node.next = head;
        head = node;
    }

    public Node pop() {
        if (head == null) {
            return null;
        }
        Node target = head;
        head = head.next;
        return target;
    }

    public Node peek() {
        if (head == null) {
            return null;
        }

        return head;
    }
}
```

```
private static Stack<Integer> mySolution(Stack<Integer> st) {
    Stack<Integer> newStack = new Stack<>();
    while (!st.isEmpty()) {
        newStack.push(st.pop());
    }
    return newStack;
}
```

```
private static void mySolution(Stack<Integer> st) {
    if (st.isEmpty()) {
        return;
    }
    Integer target = st.pop();
    mySolution(st);
    insertAtBottom(target,st);
}

private static void insertAtBottom(Integer target, Stack<Integer> st) {
    if (st.isEmpty()) {
        st.push(target);
        return;
    }

    Integer temp = st.pop();
    insertAtBottom(target,st);
    st.push(temp);
}
```

```
private static boolean check(String exp) {

    for (int i = 0; i < exp.length(); i++) {
        char target = exp.charAt(i);
        if (left.contains(target)) {
            st.push(target);
        }

        if (right.contains(target)) {
            if (st.isEmpty()) {
                return false;
            }

                return false;
            }
        }

    }
    return true;
}
```

```
private static boolean check(String exp, int targetLocation) {
    Stack<Character> st = new Stack<>();
    List<Character> left = List.of('{', '[', '(');
    List<Character> right = List.of('}', ']', ')');

    if (!left.contains(exp.charAt(targetLocation))) {
        return false;
    }

    int targetSize = -1;
    for (int i = 0; i < exp.length(); i++) {
        char target = exp.charAt(i);
        if (left.contains(target)) {
            st.push(target);
            if (i == targetLocation) {
                targetSize = st.size();
            }
        }

        if (right.contains(target)) {
            if (st.isEmpty()) {
                return false;
            }
            int index = right.indexOf(target);
            if (st.peek() != left.get(index)) {
                return false;
            }
            if (targetSize == st.size()) {
                System.out.println("여기에요 " + i);
                break;
            }
            st.pop();
        }

    }
    return true;
}
```

```
private static int check(String exp) {
    int count = 0; // 뒤집어야하는 개수
    Stack<Character> st = new Stack<>();
    for (int i = 0; i < exp.length(); i++) {
        char target = exp.charAt(i);
        if (st.size() == 0) {
            st.push(target);
            continue;
        }

        if (target == '{' && st.peek() == '}') {
            st.pop();
        } else {
            st.push(target);
        }
    }

    if (st.size() == 0) {
        return 0;
    }
    if (st.size() > 0) {
        if (st.size() % 2 == 1) {
            throw new RuntimeException(" exception");
        } else {
            count = st.size() / 2;
        }
    }

    return count;
}
```

### ps3-1
```
문제. 포스트픽스 계산하기
포스트픽스로 주어진 식을 계산하는 코드를 작성하라. 수식은 사칙연산 (+, -, *, /)만 사용한다고 가정한다.

예) 12+         => 3
예) 123+-5*  => -20

인픽스: 수식 앞 뒤로 피연산자를 두는 표기법. 예) (1 - (2 + 3)) * 5
프리픽스: 수식 뒤에 피연산자 두개를 두는 표기법. 예) *-+2315
포스트픽스: 수식을 피연산자 두개 뒤에 두는 표기법. 예) 123+-5*
```
```
시간복잡도 O(n)
공간복잡도 O(n)

public static void main(String[] args) throws IOException {
    String target = "12+";
    Stack<Integer> numbers = new Stack<>();

    for (int i = 0; i < target.length(); i++) {
        char current = target.charAt(i);
        if (current >= '0' && current <= '9') {
            numbers.push(current - '0');
        } else {
            Integer second = numbers.pop(); // 먼저 뽑은게 second가된다
            Integer first = numbers.pop();
            int result = 0;
            switch (current) {
                case '+':
                    result = (first + second);
                    break;
                case '-':
                    result = (first - second);
                    break;
                case '*':
                    result = (first * second);
                    break;
                case '/':
                    result = (first / second);
                    break;
            }
            numbers.push(result);
        }
    }

    System.out.println(numbers.pop());
}
```
